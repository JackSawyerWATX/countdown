# Countdown Timer

A simple Python console program that counts down from a user-specified number of seconds.

## Features
- Takes user input for countdown duration  
- Displays remaining time dynamically in the console  
- Alerts when the countdown reaches zero

## Technologies Used
- Python 3.x  
- Built-in `time` module

## Installation
1. Clone or download this repository.  
2. Ensure Python 3 is installed on your system.

## Usage
Run the script in your terminal:
```bash
python countdown_timer.py
```
